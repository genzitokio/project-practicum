# Кейс 1. Прогнозирование оттока клиентов банка

Бинарная классификация для предсказания оттока клиентов кредитных карт.

## Датасет

[Credit Card Customers](https://www.kaggle.com/datasets/whenamancodes/credit-card-customers-prediction)

## Результаты

- Лучшая модель: Random Forest
- Recall: 0.87
- Accuracy: 0.91

## Файлы

- `credit_card_churn.ipynb` — ноутбук с анализом и моделями
- `Credit_Card_Churn_Presentation.pptx` — презентация
- `data/BankChurners.csv` — датасет

## Запуск

```bash
pip install -r requirements.txt
jupyter notebook credit_card_churn.ipynb
```
